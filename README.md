# 👾 Go × Python AI: Reverse Bullet Hell Boss Battle

Go言語で描画される2D弾幕ゲームのボスを、外部のPython AIプロセスがリアルタイムに操作する、役割逆転型の**「超高難易度弾幕ボスバトル」**です。

---

## 🎮 Game Concept (ゲーム概要)

プレイヤーの目的は、全方位に激しい弾幕を展開し、ボスの体力を削りきること。
対するボス（AI）は、その無数の弾幕の隙間を恐ろしいフットワークで縫うように避けながら、プレイヤーへの「体当たり（接触ダメージ）」を狙ってジワジワと肉薄してきます。

AIがあなたの弾幕をすべて見切り、じりじりとプレッシャーをかけてくる絶望感とスリルを楽しんでください！

---

## 🧠 AI Features (ボスの超絶かしこい特徴)

- **ミリ秒単位の未来弾道予測**: 
  プレイヤーの弾の座標・角度（`Deg`）・速度（`Speed`）をリアルタイムに解析。数フレーム先の衝突ルートを先読みし、弾道に対して直角にステップを踏むことで紙一重の神回避を実現します。
- **壁ハメ・ハメ殺しの完全拒否**: 
  320x240の狭いステージでボスが壁際に追い詰められたことを感知すると、すべての思考をバイパスして最優先で中央の安全エリアへ離脱。プレイヤーによる壁ハメ戦術は通用しません。
- **的を絞らせない螺旋接近**: 
  ただ逃げ回るだけのAIではありません。中距離ではプレイヤーの直線弾を無効化するように周囲を高速で旋回しながら、ジワジワと内側へ螺旋を描いて切り込み、執拗に体当たりを狙ってきます。

## 📦 プレイ方法 (How to Play)

PythonやGoの環境構築は**一切不要**です。
以下の3つのファイルが同じフォルダに揃っていることを確認し、**ゲーム本体のEXEファイルをダブルクリック**するだけで即座にゲームが起動します。

### 📋 必須ファイル構成
```text
├── [ゲーム本体のEXE]  <-- これをダブルクリックして起動
├── ai.exe            <-- バックグラウンドで自動起動するAIプロセス
└── bgm.wav           <-- ゲームのBGM（WAV形式）
```

---

## 🕹️ 操作方法 (Controls)

- `W` / `A` / `S` / `D` : プレイヤーの移動
- `1` : 弾幕スキル 1（全方位15度刻み・低速・拡散弾）
- `2` : 弾幕スキル 2（全方位25度刻み・中速・高密度弾）

### 🏆 勝利条件
ボスは180フレームごとに自動で体力を回復します。ボスに捕まってプレイヤーのHP（初期値20）がゼロになる前に、ボスの圧倒的な回避AIの裏をかく弾幕パターンを構築し、5000のHPを削りきればあなたの勝利です！
